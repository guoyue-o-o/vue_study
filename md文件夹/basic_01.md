 # 2.js表达式和js语句

区分js表达式和语句：
- 1.表达式：一个表达式会产生一个值，可以放在任何一个需要值的地方(可以接收到这个值)
  - （1）a
  - （2）a + b
  - （3）demo(1) // 函数调用表达式
  - （4）三元表达式 x ? a ：b
- 2.js代码（语句）
  - （1）if(){}
  - （2）for(){}
  - （3）switch case
